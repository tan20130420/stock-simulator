# 📈 智能波段交易模拟器

基于 Streamlit + AKShare 的A股/可转债/基金波段交易策略回测工具。

## ✨ 功能特性

- **多市场支持**: A股、可转债、ETF基金、场内基金(LOF)
- **智能参数搜索**: 网格搜索最优(x,y)参数组合
- **1/3仓位管理**: 动态高抛低吸策略
- **完整手续费**: 买入佣金0.03% + 卖出佣金0.03% + 印花税0.1%
- **可视化分析**: 收益曲线、交易点标注、参数热力图
- **交易明细导出**: 支持CSV下载

## 🚀 快速开始

### 本地运行

```bash
# 1. 克隆仓库
git clone https://github.com/yourusername/stock-trading-simulator.git
cd stock-trading-simulator

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行应用
streamlit run app.py
```

### 部署到 Streamlit Cloud

1. Fork 本仓库到你的 GitHub
2. 访问 [share.streamlit.io](https://share.streamlit.io)
3. 点击 "New app" → 选择你的仓库 → 选择 `app.py`
4. 点击 Deploy，获得公开访问链接

## 📖 使用说明

1. **选择市场类型**: A股 / 可转债 / ETF基金 / 场内基金
2. **输入代码**: 6位数字代码（如000001、510300）
3. **设置日期范围**: 选择回测时间段
4. **配置参数**:
   - 初始资金: 默认10万元
   - 目标收益率: 期望达到的最低收益
   - x范围: 下跌买入幅度搜索范围
   - y范围: 上涨卖出幅度搜索范围
   - 步长: 搜索精度（默认0.5%）
5. **点击"开始回测"**: 系统自动搜索最优参数并展示结果

## ⚙️ 策略逻辑

### 1/3仓位管理法

```
资金: 100,000元 → 3份，每份33,333元

买入规则:
  - 首次买入: 起始日投入第1份
  - 下跌补仓: 价格较"上次买入价"下跌x%时，投入下1份
  - 最多持有3份（满仓）

卖出规则:
  - 上涨减仓: 价格较"平均成本"上涨y%时，卖出1份
  - 允许空仓: 卖完最后1份后等待新买入信号
```

### 手续费计算

| 操作 | 费率 |
|------|------|
| 买入 | 佣金 0.03%，最低5元 |
| 卖出 | 佣金 0.03% + 印花税 0.1% = 0.13%，最低5元 |

## 📊 输出内容

- **最优参数**: 推荐(x,y)值及预期收益
- **收益曲线**: 策略收益 vs 买入持有对比
- **交易明细**: 完整的买卖记录（可导出CSV）
- **参数热力图**: 所有参数组合的收益分布
- **风险指标**: 最大回撤、夏普比率、交易次数

## ⚠️ 风险提示

1. 本工具仅供学习研究，不构成投资建议
2. 历史回测结果不代表未来收益
3. 实际交易需考虑滑点、流动性等因素
4. 网格交易策略在单边行情中可能表现不佳
